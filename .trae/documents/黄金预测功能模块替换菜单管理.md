## 实施计划

### 一、后端开发（4个文件）

#### 1. 创建数据模型 `models/gold.py`
- 表 `gold_price`：存储黄金历史价格数据
  - 字段：id, date(日期), price_cny(人民币金价/克), price_usd(美元金价/盎司), change_pct(涨跌幅), open(开盘价), high(最高价), low(最低价), close(收盘价), volume(成交量), created_at
- 表 `gold_prediction`：存储预测记录
  - 字段：id, user_id, predict_date(预测日期), predict_price(预测价格), model_type(预测模型: ma/ema/lr/ensemble), params(JSON参数), confidence(置信度), created_at

#### 2. 创建Pydantic验证模型 `schemas/gold.py`
- `GoldPriceInfo` — 价格信息响应
- `GoldPredictionCreate` — 创建预测请求
- `GoldPredictionInfo` — 预测结果响应
- `GoldPredictParams` — 预测参数设置

#### 3. 创建路由 `routers/gold.py`
- `GET /api/gold/prices` — 获取历史价格列表（分页+日期范围筛选）
- `GET /api/gold/latest` — 获取最新金价
- `GET /api/gold/predict` — 执行黄金价格预测（支持4种模型：移动平均/指数平滑/线性回归/集成模型）
- `GET /api/gold/predictions` — 获取历史预测记录
- `GET /api/gold/params` — 获取预测参数选项
- 预测算法：基于历史数据，使用移动平均(MA)、指数平滑(EMA)、线性回归(LR)三种算法，集成模型取加权平均

#### 4. 修改 `main.py`
- 注释掉 `app.include_router(menu.router)` 并标注"已停用"
- 添加 `app.include_router(gold.router)`
- 更新 `models/__init__.py` 和 `schemas/__init__.py` 导入

### 二、前端开发（3个文件修改 + 2个文件新建）

#### 5. 创建API接口 `api/gold.ts`
- `getGoldPrices(params)` — 获取历史价格
- `getLatestPrice()` — 获取最新金价
- `predictGold(params)` — 执行预测
- `getPredictions()` — 获取预测记录
- `getPredictParams()` — 获取参数选项

#### 6. 创建页面 `pages/gold/gold.vue`
- **顶部导航栏**：紫蓝渐变背景，返回+标题"黄金预测"
- **实时金价卡片**：当前金价、涨跌幅、更新时间
- **价格走势图**：CSS实现的折线图（同天气页面温度折线图风格），支持7天/30天/90天切换
- **预测面板**：
  - 参数设置：预测天数(7/14/30)、预测模型(MA/EMA/LR/集成)
  - 预测结果：预测价格、置信度、涨跌趋势
  - 预测走势图：历史+预测拼接折线
- **历史预测记录**：列表展示

#### 7. 修改 `pages.json`
- 保留 `pages/menu/menu` 路由注册（不删除）
- 新增 `pages/gold/gold` 路由注册

#### 8. 修改 `home.vue`
- 将快捷操作中 `{ icon: '📋', label: '菜单管理', bgColor: '#fff3e0', action: 'menu' }` 替换为 `{ icon: '🥇', label: '黄金预测', bgColor: '#fff8e1', action: 'gold' }`
- 在 `handleQuickAction` 中添加 `gold` 的跳转逻辑，注释掉 `menu` 的跳转

### 三、设计原则
- 严格遵循项目现有代码风格（Vue 3 Composition API + TypeScript、FastAPI + SQLAlchemy + Pydantic）
- 导航栏渐变色风格与天气/新闻页面一致（紫蓝渐变）
- 统一响应格式 `{ code, message, data }`
- 预测算法纯Python实现，不引入额外依赖
- 前端图表用CSS实现（与天气页温度折线图同风格），不引入第三方图表库
